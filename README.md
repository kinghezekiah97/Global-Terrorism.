# Global & Regional Risk Intelligence: A Spatiotemporal Analysis of Security Threats
This project provides a comprehensive analysis of the Global Terrorism Database (GTD), focusing on identifying high-risk geographic zones, seasonal attack patterns, and tactical shifts from 1970 to 2017. The analysis culminates in a high-resolution deep dive into the Nigerian security landscape to assist in local Business Continuity Planning (BCP).
Technical Stack

    Language: Python

    Libraries: Pandas (Data Wrangling), NumPy (Numerical Analysis), Matplotlib / Seaborn (Statistical Visualization)

    Dataset: 181,000+ records with 135 features.

Key Analytical Pillars

    Global Threat Mapping: Identifying the top 10 most affected nations to assist in international market-entry risk assessment.

    Tactical Trend Analysis: Quantifying the most common attack types (e.g., Bombings vs. Armed Assaults) to inform physical security infrastructure requirements.

    Seasonality Detection: Using time-series analysis to identify monthly peaks in activity (March, May, and July), enabling predictive resource allocation.

    Regional Deep Dive (Nigeria): * Isolated and analyzed 2,000+ incidents.

        Identified "Private Citizens & Property" as the primary target sector (exceeding military and government targets).

        Visualized the dominance of specific non-state actors (Boko Haram, Fulani extremists) to understand regional operational risks.

Business Value Generated

    Security ROI: Data-driven justification for security budget allocation based on historical attack vectors.

    Logistics Optimization: Identifying "low-risk months" for high-value asset movement in volatile regions.

Targets:
Public places, government buildings, embassies, transportation systems.

Sometimes symbolic targets (e.g., World Trade Center).

Actors:
Organized groups (e.g., Al-Qaeda, ISIS, Boko Haram).

State sponsors (in indirect forms).
